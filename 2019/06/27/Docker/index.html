Docker

2019-06-27

容器的典型生命周期：创建、管理、停止、删除

docker run

-i 保证容器中的STDIN是开启的
-t 为要创建的容器分配一个伪tty终端
–name 为容器指定一个名称
ubuntu 基于ubuntu镜像来创建容器
bin/bash 启动Bash shell

docker start

重新启动已经停止的容器

docker ps

查看正在运行的docker
-a 查看所有docker

docker rm

删除容器

docker image

列出docker镜像
